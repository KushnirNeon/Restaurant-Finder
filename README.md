# Restaurant Finder

A simple web application to display restaurants on a map, filter by cuisine type, and add new restaurants dynamically. Built using **HTML, CSS, JavaScript (Leaflet.js)** for the frontend and **Node.js + Express + MongoDB** for the backend.

---

## Features

- Interactive map showing restaurant locations using Leaflet.js
- Filter restaurants by cuisine type
- Add new restaurants with details: name, description, rating, cuisine type, latitude, and longitude
- Backend API built with Express.js and MongoDB for storing restaurant data
- Dynamic updates on the map when adding restaurants
